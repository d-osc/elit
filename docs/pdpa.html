สรุปแบบย่อ

สิ่งที่เว็บไซต์นี้เก็บ

เว็บไซต์นี้เก็บข้อมูลการตั้งค่า theme ใน localStorage เพื่อจำการแสดงผลที่คุณเลือก และอาจมีข้อมูลทางเทคนิคมาตรฐานจากแพลตฟอร์มโฮสต์ เช่น IP address, user agent, และเวลาเข้าใช้งาน เพื่อความปลอดภัยและการดำเนินงานของระบบ

สิ่งที่เว็บไซต์นี้ไม่ทำ

เว็บไซต์นี้ไม่มีระบบสมัครสมาชิก, ไม่มีแบบฟอร์มรับข้อมูลส่วนบุคคลโดยตรง, และไม่ได้ใช้ localStorage เพื่อเก็บข้อมูลอ่อนไหวหรือข้อมูลระบุตัวตนของผู้ใช้งาน

บริการภายนอก

เมื่อคุณกดลิงก์ไป GitHub, npm หรือเว็บไซต์ภายนอกอื่น การประมวลผลข้อมูลจะเป็นไปตามนโยบายของผู้ให้บริการปลายทางนั้น ๆ

1. ข้อมูลที่อาจมีการประมวลผล

  • ข้อมูลการตั้งค่าฝั่งเบราว์เซอร์ เช่น theme ที่เก็บไว้ภายใต้ localStorage key ของเว็บไซต์นี้
  • ข้อมูลทางเทคนิคที่แพลตฟอร์มโฮสต์หรือ reverse proxy อาจบันทึกไว้ตามปกติ เช่น IP address, user agent, referrer, request path และเวลาที่ร้องขอ
  • ข้อมูลจากการโต้ตอบที่เกิดจากการกดปุ่มคัดลอก ซึ่งใช้ clipboard API ของเบราว์เซอร์เฉพาะตอนที่ผู้ใช้กดเอง โดยเว็บไซต์นี้ไม่ได้จัดเก็บข้อความดังกล่าวไว้บน server ของตัวเอง

2. วัตถุประสงค์ในการใช้ข้อมูล

  • เพื่อแสดงผลเว็บไซต์และเอกสารทางเทคนิคให้ทำงานได้อย่างถูกต้อง
  • เพื่อจดจำการตั้งค่าการแสดงผลที่ผู้ใช้เลือก เช่น dark หรือ light theme
  • เพื่อความมั่นคงปลอดภัย การตรวจสอบปัญหาระบบ และการป้องกันการใช้งานที่ผิดปกติในระดับโฮสต์หรือโครงสร้างพื้นฐาน

3. ฐานทางกฎหมาย

การประมวลผลข้อมูลตามประกาศนี้อาศัยฐานทางกฎหมายตามความจำเป็นในการให้บริการเว็บไซต์, ประโยชน์โดยชอบด้วยกฎหมายในการรักษาความปลอดภัยและเสถียรภาพของระบบ, และความยินยอมของผู้ใช้ในกรณีที่กฎหมายกำหนดให้ต้องขอโดยเฉพาะ

4. การเปิดเผยข้อมูลให้บุคคลภายนอก

ข้อมูลอาจถูกประมวลผลผ่านผู้ให้บริการโฮสต์, CDN, หรือแพลตฟอร์มภายนอกที่ใช้สำหรับเผยแพร่เว็บไซต์นี้ตามความจำเป็นทางเทคนิค โดยเมื่อคุณออกจากโดเมนนี้ไปยัง GitHub, npm หรือเว็บอื่น ๆ จะอยู่ภายใต้นโยบายของผู้ให้บริการปลายทางนั้นโดยตรง

5. ระยะเวลาการเก็บรักษา

  • ข้อมูล theme preference จะคงอยู่ในเบราว์เซอร์ของคุณจนกว่าจะล้าง localStorage หรือเปลี่ยนค่าด้วยตัวเอง
  • ข้อมูล log ระดับโฮสต์จะขึ้นอยู่กับการตั้งค่าของผู้ให้บริการโฮสต์หรือระบบเครือข่ายที่เกี่ยวข้อง

6. สิทธิของเจ้าของข้อมูล

ภายใต้ PDPA คุณอาจมีสิทธิในการเข้าถึง, ขอรับสำเนา, ขอแก้ไข, ขอให้ลบ, ขอระงับการใช้, คัดค้านการประมวลผล, ถอนความยินยอม และร้องเรียนต่อหน่วยงานกำกับดูแล ตามเงื่อนไขที่กฎหมายกำหนด

7. ช่องทางติดต่อ

หากมีคำถามเกี่ยวกับประกาศนี้หรือมีคำขอเกี่ยวกับข้อมูลส่วนบุคคล กรุณาติดต่อผ่านช่องทางที่เผยแพร่ใน GitHub repository ของโครงการ Elit หรือช่องทางติดต่ออื่นที่ผู้ดูแลโครงการประกาศไว้ในเวลานั้น
